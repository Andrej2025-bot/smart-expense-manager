# Smart Expense Manager (CLI App)

A simple command-line application written in Python to manage personal expenses.

## Features

- Add new expense (name, amount, category)
- Display all expenses with index
- View statistics:
  - Total amount
  - Number of expenses
  - Smallest expense
  - Largest expense
- Delete expense by index with confirmation
- Input validation for all user entries

## Technologies

- Python 3
- Lists
- Dictionaries
- Loops (while, for)
- Conditionals
- Exception handling (try/except)

## How to Run

```bash
python smart_expense_manager.py
